Health Gauge

Options

sensitivity number
Controls the pointer animation speed, the higher the sensitivity, the higher the speed. Values are in the range of 0 to 20, where the maximum value will result in immediate response, i.e. without transition animation.
val number
Sets the value for the gauge, in the range of 0 to 100.
colors object
Configures the color theme by providing an object with the following properties, as CSS color values:
border string
The border color.
tick string
The tick color.
background array(2)
The background gradient colors, bottom-up.
basis array(2)
The basis gradient colors, bottom-up.
pointer array(3)
The pointer colors, in the following order: border, left-gradient, right-gradient.
dial array(5)
The dial gradient colors, left to right.
